## ANALYSE DE LA PERFORMANCE DES VENTES
Cette application vise à afficher un tableau de bord à partir de données fournies par un back-end construit avec Spring Boot.

## Technologies utilisées
- TypeScript;
- HTML;
- CSS;
- Java;
- React;
- React Router;
- Bootstrap;
- Axios;
- Spring Boot;
- PostgreSQL;
- Heroku;
- Netlify.
